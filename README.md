# Dependencies 
[Logger](https://github.com/darkFrancis/Logger)
Add Conan remote https://darkfrancisdod.jfrog.io/artifactory/api/conan/public-conan

# Build
```bash
cd path/to/ProjectManager
mkdir build
cd build
cmake ..
make
sudo cmake --install .
```
